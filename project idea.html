My Project Idea

Overview

My idea is to turn a raspberry pi into a cheaper affordable way to turn a normal tv into a smart tv, when you plug the raspberry pi into the back of a TV like a Chromecast, Usb or some other peripheral using HDMI or Usb slots, when the pi is inserted into the TV it runs webOS (operating system) using the open-source LG webOS application using the raspberry pi. It also connects to your TV remote to the raspberry pi "Smart device", by using the Simplink button on your tv remote. Fully capable Smart TV just by using a modded Raspberry pi 4 plugged into the back of a non smart TV.

Description

My product features the use of a Raspberry pi 4, running the latest version of the WebOS software while also having applications such as Netflix, YouTube and other media applications while also having internet connectivity, to make sure the streaming apps work properly as they need to be connected to the internet and be able to keep up to date with updates for the streaming apps. I want my device to be something like a Chromecast that you plug into the back of a tv using the HDMI or Usb port. I would also like to add a case to the raspberry pi 4,so that it doesn’t look like microchips hanging from the back of the TV unit. I hope when the “Smart Device” is turned on is that, when you change the TV to one of the HDMI ports, it boots up WebOS, loads the apps like Netflix, YouTube etc. I also hope to have a application store in case the applications you want aren’t on the “Smart device” can be downloaded and ready to use. When you boot up the “Smart device” for the first time, you get asked to connect your TV remote to the "Smart device" by pressing the "simplink" button on the tv remote, then it will connect to the Smart device allowing you to use the device with your own TV remote. I would like to have web browser supoort from one of the application widgets,allowing you use the internet and look up information.
